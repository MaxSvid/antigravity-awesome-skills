**Skills** are small markdown files that teach AI how to do these specific tasks perfectly, every time.

These skills follow the universal **SKILL.md** format and work with any AI coding assistant that supports agentic skills.

| Tool            | Type | Invocation Example                | Path                                                                  |
| :-------------- | :--- | :-------------------------------- | :-------------------------------------------------------------------- |
| **Claude Code** | CLI  | `>> /skill-name help me...`       | `.claude/skills/`                                                     |
| **Gemini CLI**  | CLI  | `(User Prompt) Use skill-name...` | `.gemini/skills/`                                                     |

> [!TIP]
> **Universal Path**: recommend cloning to `.gemini/skills/` or `.claude/skills/`. Most modern tools (Antigravity, recent CLIs) look here by default.
